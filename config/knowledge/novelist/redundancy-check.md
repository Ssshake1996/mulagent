# 冗余描述检测与优化指南

长篇小说中，重复的描述会让读者感到乏味。本指南帮助识别和消除冗余。

---

## 六类常见冗余

### 1. 外貌重复描写

同一角色的外貌在多章反复以相同方式描述。

| 冗余写法 | 优化策略 |
|---------|---------|
| 每次出场都写"她的长发如瀑布般垂落" | 首次完整描写，之后用"她撩了一下头发"等动态细节 |
| 每章都提"他那双深邃的眼睛" | 改为通过他的眼神表达情绪："他的目光变得锐利" |
| 反复描述服装细节 | 只在服装变化或有剧情意义时描写 |

**描写预算**：
- 完整外貌描写：每角色每卷最多**2次**
- 单一特征提及：每角色每篇章最多**3次**
- 之后用动作、习惯、他人反应代替

### 2. 场景雷同描写

同一地点反复使用相同的环境描写。

| 冗余写法 | 优化策略 |
|---------|---------|
| 每次进办公室都写"宽敞明亮的办公室" | 第二次起用不同感官："咖啡的余味还留在空气里" |
| 每次写森林都是"参天大树、鸟鸣啁啾" | 换角度：通过脚下落叶的质感、光线变化来写 |
| 夜晚场景总是"月光如水" | 尝试：无月的黑暗、灯光、星空等变体 |

**原则**：同一场景的第二次描写，必须用不同的感官通道或不同的情绪滤镜。

### 3. 情绪反应套路化

角色对不同事件总是用相同的方式表达情绪。

**高频套路词（出现3次以上即为过度）：**

| 套路词 | 替代方案（至少准备5种） |
|--------|---------------------|
| 心中一震 | 胃像被人攥了一把 / 后背窜过一道寒意 / 脚步顿住 / 指尖发麻 / 瞳孔骤缩 |
| 不由得 | [直接写动作，删除这个词] |
| 嘴角微微上扬 | 咧开嘴 / 眼尾的皱纹加深了 / 低头笑了一声 / 嗤了一声 / 鼻子哼了哼 |
| 眼中闪过一丝XX | 眉头拧了一下 / 下意识攥紧了衣角 / 呼吸停了半拍 / 牙关咬紧 / 喉结滚动了一下 |
| 深吸一口气 | 用力揉了把脸 / 把手插进头发里 / 靠在墙上闭了会儿眼 / 灌了一口凉水 / 使劲捏了捏鼻梁 |
| 心如刀绞 | 胸口像压了块石头 / 鼻腔一阵酸涩 / 视线模糊了 / 嘴唇咬出了血 / 指甲掐进掌心 |
| 浑身一颤 | 手抖了一下 / 膝盖发软 / 后退了半步 / 杯子差点脱手 / 声音哑了 |
| 目光如炬 | 盯得人头皮发麻 / 那种眼神让人想后退 / 像在称量什么 |

### 4. 过渡语重复

章节之间、场景之间总用相同的衔接方式。

| 套路过渡 | 替代方案 |
|---------|---------|
| "就在这时" | 直接写发生的事，不用过渡 |
| "话说回来" | 用角色的动作或想法自然转场 |
| "与此同时" | 切到另一个角色的视角直接开始 |
| "时光荏苒" | 用具体细节暗示时间流逝："桌上的咖啡已经凉透了" |
| "翌日清晨" | "闹钟响的时候，她已经醒了两个小时" |

### 5. 能力/设定重复解释

担心读者忘记而反复解释同一个设定或能力。

**原则**：
- 首次出现：完整解释
- 第二次：用一句话提示即可
- 第三次及之后：直接使用，不再解释。相信读者的记忆力。
- 如果间隔超过100章再次出现，可以通过对话自然带出，但不要照搬原文

### 6. 描述堆砌

同一段中用多个近义词/形容词描述同一件事。

| 堆砌写法 | 精简写法 |
|---------|---------|
| "巨大的、庞大的、硕大无朋的身躯" | "那东西挡住了半边天" |
| "愤怒的、暴躁的、怒不可遏的情绪" | "他一脚踢翻了椅子" |
| "美丽的、动人的、令人心醉的容颜" | "他看了一眼，就没能再移开目光" |
| "寒冷的、刺骨的、冰冷彻骨的风" | "风一吹，嘴唇立刻裂开了" |

**判断标准**：如果删掉一个形容词意思不变，就该删掉。

## 冗余检查节奏

| 时机 | 检查范围 | 方法 |
|------|---------|------|
| 每章写后润色 | 本章内 | 人工检查本章是否有上述六类冗余 |
| 每10章 | 最近10章 | 运行 `detect_redundancy.py` 脚本 |
| 每篇章结束 | 整个篇章 | 运行脚本 + 人工审查高频词报告 |
| 每卷结束 | 整卷 | 全面冗余审计 |

---

## 优化原则

1. **首详后略** — 第一次浓墨重彩，之后点到为止
2. **换角度不换信息** — 同一事物用不同感官/视角描写
3. **用动作替代形容** — "他很紧张"改为"他的手在抖"
4. **信任读者** — 不要低估读者的记忆力和理解力
5. **删比加好** — 如果犹豫要不要保留一段描写，删掉它
