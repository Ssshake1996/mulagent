# 标点符号规范

中文小说的标点符号直接影响阅读节奏和专业感。

---

## 基础规范

### 对话标点

| 场景 | 正确用法 | 错误用法 |
|------|---------|---------|
| 陈述 | "我知道了。"他说。 | "我知道了"他说。 |
| 疑问 | "你确定？"她问。 | "你确定?"她问。 |
| 感叹 | "住手！"他大喊。 | "住手!"他大喊。 |
| 中断 | "我本来想——" | "我本来想..." |
| 犹豫 | "我……我不知道。" | "我...我不知道。" |
| 对话中动作 | "走吧，"他站起来，"时间不多了。" | "走吧"他站起来"时间不多了" |

**关键规则：**
- 中文对话使用双引号 `""` ，引号内嵌套使用单引号 `''`
- 对话结尾的标点放在引号**内**
- 对话标签（他说/她问）前用逗号，后用句号
- 中文使用中文标点（全角），不混用英文标点（半角）

### 省略号

| 正确 | 错误 | 说明 |
|------|------|------|
| …… | ... | 中文省略号是两个三点，共六个点 |
| "我……" | "我..." | 对话中的省略也用中文省略号 |

**使用场景：**
- 语言中断、话没说完："如果我们能……"
- 沉默、停顿："他沉默了很久。……'好吧。'"
- 列举省略：刀、枪、剑……各种兵器

**过度使用警告：** 省略号不应在3段内出现超过2次，否则显得犹豫不决。

### 破折号

| 正确 | 错误 | 说明 |
|------|------|------|
| —— | -- | 中文破折号占两个字符位 |
| ——是他。 | -是他。 | 不要用短横线代替 |

**使用场景：**
- 解释说明："那个人——她的前夫——突然出现了。"
- 话被打断："我刚要说——" "闭嘴！"
- 语气转折："一切都很好——至少表面上如此。"

### 顿号 vs 逗号

| 使用顿号 | 使用逗号 |
|---------|---------|
| 并列词语：刀、枪、剑、戟 | 并列短语：他拿起枪，推开门，冲了出去 |
| 并列形容词：高大、魁梧的身影 | 分句之间：他站起来，看了她一眼 |

## 常见标点错误

### 1. 中英文标点混用

这是最常见的问题。全文必须统一使用中文标点。

| 中文标点（正确） | 英文标点（错误） |
|----------------|----------------|
| ， | , |
| 。 | . |
| ！ | ! |
| ？ | ? |
| ：| : |
| ；| ; |
| "" | "" |
| '' | '' |
| —— | -- |
| …… | ... |
| （）| () |

### 2. 标点缺失

| 错误 | 正确 |
|------|------|
| "走吧"他说"我们没时间了" | "走吧，"他说，"我们没时间了。" |
| 她转身走了他没有追 | 她转身走了。他没有追。 |

### 3. 标点重复/堆砌

| 错误 | 正确 |
|------|------|
| "什么？！" | "什么？" 或 "什么！" |
| "不！！！" | "不！" |
| "真的吗？？" | "真的吗？" |
| 他说……。 | 他说…… |

### 4. 句号和逗号滥用

| 问题 | 说明 |
|------|------|
| 全是逗号的长句 | 超过3个逗号的句子应拆分为短句 |
| 全是句号的短句 | 连续5个以上短句显得单调，适当合并 |
| 叙述全用逗号连接 | 不同事件/动作之间用句号分隔 |

## 标点与节奏

标点符号是控制阅读节奏的工具：

### 快节奏（紧张、动作场面）

```
短句。句号分隔。制造紧迫感。

示例：
枪响了。他扑倒在地。血从肩膀渗出来。来不及想了。他翻身，滚到柱子后面。
```

### 慢节奏（内心、情感场面）

```
长句用逗号连接，舒缓的节奏，给读者时间感受。

示例：
她站在窗前，看着外面的雨一滴一滴落在玻璃上，想起了很多年前那个同样下着雨的下午，他第一次牵住她的手。
```

### 戏剧性停顿

```
用破折号或省略号制造停顿：

示例：
"凶手就在这个房间里。"他扫视众人。"就是——"
所有人屏住呼吸。
"你。"
```

## 标点符号检查清单

每章完成后逐项检查：

- [ ] **全文使用中文标点**，无英文标点混入
- [ ] **对话标点完整**，引号、逗号、句号位置正确
- [ ] **省略号格式正确**，使用"……"而非"..."
- [ ] **破折号格式正确**，使用"——"而非"--"或"-"
- [ ] **无标点重复/堆砌**，无"？！""！！！"
- [ ] **逗号不过载**，单句逗号不超过3个
- [ ] **省略号不过密**，3段内不超过2个
- [ ] **标点服务于节奏**，快场面用短句+句号，慢场面用长句+逗号
